## [0.1.0] - 2026-05-25

### Added
- Initial release of Splitism
- Add people to a shared ledger
- Record expenses with payer, amount, date, and description
- Multiple payers per expense
- Equal and exact split modes
- Balance overview with net amounts per person
- Settlement suggestions (minimal payment set)
- Multi-currency support: USD, LKR, EUR, GBP, AUD, CAD, INR, JPY
- CSV export and import with full ledger roundtrip
- localStorage persistence with schema versioning and migration (v1 → v2 → v3)
- Mobile-first responsive UI
- Overview, Activity, People, and Data views
